# Sistema de Tickets - LTHS Tecnologia

Sistema interno de tickets desenvolvido para registro, organização e acompanhamento de solicitações de suporte.

## Sobre o projeto

O **Sistema de Tickets** foi desenvolvido pela **LTHS Tecnologia** com o objetivo de centralizar chamados internos, facilitar o registro de problemas e melhorar o acompanhamento das demandas de suporte.

A aplicação permite cadastrar tickets com nome, e-mail, descrição do problema e nível de urgência, exibindo os registros em uma interface moderna, responsiva e intuitiva.

## Funcionalidades

- Cadastro de tickets
- Definição de prioridade
- Listagem dinâmica dos chamados
- Exclusão de tickets
- Armazenamento local com `localStorage`
- Layout responsivo para desktop e mobile
- Identidade visual personalizada com a marca **LTHS Tecnologia**

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS3
- HTML5
